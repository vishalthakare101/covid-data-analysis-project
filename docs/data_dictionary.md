| Column Name | Description |
| --- | --- |
| **Jurisdiction_Residence** | State or region where the deaths were recorded |
| **data_period_start** | Start date of the reporting period (string → converted to DATE) |
| **data_period_end** | End date of the reporting period (string → converted to DATE) |
| **COVID_deaths** | Number of COVID‑19 deaths in the period |
| **crude_COVID_rate** | Death rate per 100,000 population |
| **footnote** | Notes on suppressed values or special conditions |
| **Group** | Reporting frequency (Weekly / Monthly) |
| **pct_change_wk** | Percent change from previous week (string → converted to FLOAT) |

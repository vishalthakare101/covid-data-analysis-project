# COVID‑19 Data Analysis Project (SQL + Power BI + PPT)

## 🔍 Objective
Analyze COVID‑19 mortality data using SQL to uncover trends, patterns, and jurisdiction‑level insights.  
This project demonstrates SQL proficiency through filtering, grouping, aggregations, date handling, CTEs, and analytical reporting.

## 📘 Dataset Overview
The dataset contains COVID‑19 mortality data with:
- Jurisdiction (state/region)  
- Reporting period start & end dates  
- COVID deaths  
- Crude COVID rate  
- Footnotes (suppressed values, notes)  
- Weekly/monthly reporting groups  

Dataset source: Public CDC COVID mortality data  
*(Dataset not included due to size and licensing.)*

---

## 🧾 SQL Topics Covered
- SELECT, WHERE, ORDER BY  
- DISTINCT  
- TOP N queries  
- Aggregations (SUM, AVG, MIN, MAX)  
- GROUP BY + HAVING  
- Date conversions using `TRY_CAST`  
- CTEs  
- CROSS JOIN  
- Filtering non‑empty footnotes  
- Latest/earliest date per jurisdiction  

---

## 📊 Key Insights (Summary)
- Identified jurisdictions with highest total deaths  
- Found earliest and latest reporting periods  
- Calculated average deaths and compared jurisdictions  
- Analyzed crude COVID rate trends  
- Extracted rows with meaningful footnotes  
- Identified jurisdictions exceeding average mortality  

(Full insights in `docs/insights_summary.md`)
